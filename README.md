# Bouillon Baron - Site Web Officiel

Site web officiel du restaurant **Bouillon Baron**, établissement traditionnel sélection Michelin à Angers (49).

## 📋 Spécifications

- **Restaurant** : Bouillon Baron (Angers)
- **Statut** : Sélection Michelin confirmée (2023)
- **Évaluation** : 4,8/5 (Google)
- **Cible** : Clientèle locale + touristique (800 000 visiteurs/an)

## 🎯 Objectifs

✅ Créer un site web officiel haut de gamme  
✅ Mettre en place un système de réservation en ligne  
✅ Optimiser le SEO local et touristique  
✅ Générer ROI via augmentation réservations directes  

## 📄 Pages & Fonctionnalités

| Page | Fonctionnalités |
|------|-----------------|
| **Accueil** | Hero section, Michelin badge, Chef, Menu preview, Avis clients |
| **Restaurant** | Histoire, Philosophie, Chef & équipe, Galerie photos |
| **Carte & Menu** | Entrées/Plats/Desserts, Menus 2/3 plats, PDF téléchargeable |
| **Réservation** | Calendrier dynamique, Créneaux, Confirmation email |
| **Infos Pratiques** | Adresse, Horaires, Transports, Capacité, Privatisation |
| **Contact** | Formulaire sécurisé, Google Maps, Téléphone cliquable |
